# Visão Geral

## Objetivo

A API Gestão Financeira é um sistema backend desenvolvido em Java 21 com Spring Boot 3.2.5, responsável pelo gerenciamento de:

- Clientes
- Contas bancárias
- Investimentos
- Produtos financeiros
- Importação e exportação de dados
- Integrações externas (BrasilAPI e MockBank)

O sistema simula um núcleo básico de operações bancárias com regras de negócio transacionais e integração com serviços externos.

---

## Stack Tecnológica

- Java 21
- Spring Boot 3.2.5
- Spring Data JPA
- PostgreSQL 15
- Flyway (versionamento de banco)
- Docker / Docker Compose
- WebClient (WebFlux)
- Apache POI (Excel)
- Springdoc OpenAPI
