# Projeto Backend - Gestão Financeira

Projeto desenvolvido com foco em arquitetura limpa, boas práticas e integração externa.

## Destaques Técnicos

- Java 21
- Spring Boot 3
- PostgreSQL
- Docker
- Flyway
- OpenAPI
- Integração com APIs externas
- Regras de negócio encapsuladas no domínio
- Controle transacional
- Tratamento global de exceções

## Diferenciais

- Conversão automática de investimentos para USD
- Validação de CEP via API externa
- Importação e exportação Excel
- Simulação de saldo externo

Projeto estruturado para evolução para ambiente corporativo real.
