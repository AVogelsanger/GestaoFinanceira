# Gestão Financeira API - Resumo

API REST desenvolvida em Spring Boot para simular operações bancárias:

- Cadastro de clientes
- Abertura de contas
- Depósito e saque
- Investimentos com conversão USD
- Integração com BrasilAPI
- Exportação/Importação via Excel

Arquitetura:
Controller → Service → Repository

Banco:
PostgreSQL + Flyway

Execução:
Local ou via Docker
